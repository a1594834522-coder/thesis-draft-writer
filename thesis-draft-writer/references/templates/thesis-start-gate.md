# Thesis Start Gate

Use this as the first-screen operating checklist for thesis tasks.

## Non-Negotiable

1. Proposal challenged if the thesis path comes from a proposal or task book.
2. Platform feasibility checked before committing to an experiment route.
3. Core papers fully read before they shape method, experiment, or claim decisions.
4. Real experiment artifact produced before full-draft or final-`docx` delivery on experiment-capable topics.
5. Real reviewer subagents dispatched for independent sidecar review work before claiming that panel-style review happened.

## Immediate Stop Signals

- about to write a full thesis draft with no real experiment artifact
- about to cite a paper that was only skimmed
- about to say "reviewed" with no actual reviewer subagents for a sidecar review task
- about to accept the proposal path without challenging feasibility or scope

## Default Opening Order

1. Proposal challenge
2. Platform check
3. Core-paper reading
4. Experiment route lock
5. Real implementation and experiment work
6. Sidecar reviewer subagents when useful
7. Full-manuscript integration
