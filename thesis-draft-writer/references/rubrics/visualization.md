# Visualization Rubric

- Data figures must come from real result artifacts.
- Captions should state what the figure shows and why it matters.
- Axis labels and units must be explicit when applicable.
- Concept diagrams can start as prompt placeholders, but they must have a thesis-specific purpose.
- Prefer color-safe palettes and avoid default plotting colors when producing final thesis figures.
- For Python-generated figures, prefer 450 DPI PNG export plus SVG export for reusable final delivery.
- Figure filenames and captions should map cleanly to chapter placement and claim support.
