# Method Rubric

- Every experiment should map to a research question.
- Metrics, baselines, and comparisons should be explicit.
- Results should trace back to scripts, commands, or source data.
- If evidence is only from simulation or public data, say so plainly.
- When statistical claims are made, state the test choice and whether key assumptions were checked.
- Prefer effect sizes and confidence intervals over p-value-only reporting.
- Distinguish confirmatory analysis from exploratory analysis when both appear.
